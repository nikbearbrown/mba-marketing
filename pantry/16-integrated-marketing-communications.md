## Core mechanisms

- **Promotion mix (six elements):** advertising / PR / sales promotion / personal selling / direct marketing / digital and social.
- **Communication process (seven elements + noise):** sender → encoding → message → media → receiver → decoding → feedback. Marketer controls first four.
- **IMC** = coordinating multiple promotion mix elements around a consistent message.
- **6-step IMC planning process:** identify audience → determine objectives → design message → select channels → determine budget → measure.
- **5A customer journey:** Aware → Appeal → Ask → Act → Advocate.

## Vocabularies

- **SMART objectives** (referenced from Ch 3) for IMC campaigns.
- **Reach vs. depth** trade-off across promotion mix elements.
